O que é HTML.

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web

Paragráfo (p)

Estilos de inserção de texto

São basicamente uma forma de formatar seu texto,deixando mais visível e organizado

Lista ordenada e não ordenada

São tags (Ul - Que se trata de lista não ordenada), (Ol - Que se trata de uma lista ordenada) que ajudam a organizar itens,menus,blogs,categprias. Formando uma lista quando aplicada

Links

São tags inseridas para colocar links em determinados locais,levando o usúario ao endereço desejado
